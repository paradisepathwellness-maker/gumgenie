## PROJECT SPECIFICATIONS

### 1. FILE & EXPORT SPECIFICATIONS

#### PDF Technical Requirements
- **Format**: PDF 1.4 (universal compatibility)
- **Resolution**: 300 DPI (high resolution for handwriting)
- **Color space**: RGB (optimized for iPad display, not print)
- **Compression**: Moderate (readable + reasonable file size < 50MB)
- **Color mode**: RGB, no CMYK (screens, not print)
- **Encryption**: Optional password protection
- **Interactivity**: Hyperlinks optional (non-essential)
- **Fonts**: Embedded (no missing fonts)

#### Page Specifications
```
Paper size: 11" x 8.5" (Tabloid/Ledger landscape)
Why landscape: Better for iPad Pro, more writing space
Top margin: 0.5"
Bottom margin: 0.5"
Left margin: 0.5"
Right margin: 0.5"

Usable area: 10" x 7.5" (writing space)
```

#### Why These Specifications
- **300 DPI**: Sharp handwriting, no pixelation with stylus
- **Landscape**: Better use of iPad Pro screen (minimize scrolling)
- **11x8.5"**: Standard tabloid size, optimal for annotation apps
- **Embedded fonts**: No font substitution issues on iPad
- **RGB color**: Better on iPad screens (white is brighter than print)

### 3. TYPOGRAPHY SPECIFICATIONS

#### Font Selections
- **Heading Font**: System font or web-safe sans-serif
  - Options: Helvetica Neue, Segoe UI, San Francisco
  - Why: Notion-style, clean, professional
  - Note: Avoid custom fonts (may not render on iPad)

- **Body Font**: Same as heading
  - Size: 12pt (default, readable on iPad)
  - Weight: Regular (400)
  - Line height: 1.5

- **Small Text**: Same font
  - Size: 10pt (for instructions, captions)
  - Weight: Regular (400)

### 11. ACCESSIBILITY SPECIFICATIONS

#### Color Contrast
```
Text vs. background must meet WCAG AA (4.5:1 minimum)

Example contrast ratios:
#1F2937 (dark gray) on #FFFFFF (white) = 13.6:1 ‚úì (excellent)
#6B7280 (medium gray) on white = 7.5:1 ‚úì (good)
#9CA3AF (light gray) on white = 4.8:1 ‚úì (barely passes)
#BCC0CA (lighter gray) on white = 2.9:1 ‚úó (fails, too light)

Rule: Dark text on light background works best
```

#### Font Sizing
```
Minimum: 10pt (captions only)
Standard: 12pt (body text, labels, headers - readable)
Large: 14pt+ (main headers, titles)
Avoid: 8pt or smaller (inaccessible)
```

#### Icon/Emoji Accessibility
```
‚úì GOOD: "‚úì Complete this task"
‚úó BAD: Just "‚úì" with no label

‚úì GOOD: "‚ö° High priority"
‚úó BAD: Just "‚ö°" alone

Always pair icon with text explanation
```

#### Readability
```
Line length: Max 5 inches per line (easier to read)
Line spacing: 1.5 or more (comfortable reading)
Margins: 0.5" minimum (not cramped)
Columns: 1-2 columns max (avoid narrow text)
```
